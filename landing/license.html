03 Third-party components

System Load reads every metric using only Apple's own system frameworks — there are no third-party libraries in the metrics engine. The single external dependency in the whole project is the auto-update framework:

Sparkle
Software-update framework, added via Swift Package Manager. Licensed under the MIT License. © Sparkle Project contributors. github.com/sparkle-project/Sparkle

Because Sparkle is also MIT-licensed, the entire distributed app stays under permissive terms. Sparkle bundles a few small components (e.g. a binary-delta tool) under other permissive licenses; their notices ship inside the framework.

04 Name & icon

The MIT grant covers the code. The “System Load” name and the app icon are project marks — you're welcome to fork the source and ship your own build, but please give it a different name and icon so users aren't confused about which one is the original.
